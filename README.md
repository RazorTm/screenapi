# screenapi

## Что реализовано

- стартовый белый экран с кнопкой `Открыть`
- открытие стандартного Flutter `showModalBottomSheet`
- кастомный bottom sheet по макету со скрина
- заголовок `Выберите фотографии`
- первый ряд с тремя карточками действий:
  - `Камера`
  - `Галерея`
  - `Файл`
- отображение фотографий устройства в виде сетки
- выбор изображений по нажатию
- индикатор выбора в правом верхнем углу каждой карточки
- при выборе индикатор становится синим
- внутри синего индикатора показывается номер по порядку выбора
- возможность снять выбор повторным нажатием
- фото, сделанное камерой, добавляется в сетку
- изображения, выбранные через галерею, добавляются в сетку
- файл, выбранный через файловый менеджер, добавляется в сетку
